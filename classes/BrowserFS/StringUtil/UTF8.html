    Class: BrowserFS.StringUtil.UTF8

    Defined in: src/core/string_util.coffee

    Overview

    String utility functions for UTF-8. Note that some UTF-8 strings cannot be expressed in terms of JavaScript UTF-16 strings.

    See also:

    • http://en.wikipedia.org/wiki/UTF-8

    + (Number) str2byte(buf, str, offset, length)

    Converts a JavaScript string into unsigned bytes representing a UTF-8 string, and writes those bytes into the given buffer.

    We assume that the offset / length is pre-validated.

    Parameters:

    • buf (BrowserFS.node.Buffer) — the buffer to write into
    • str (String) — the string that will be converted
    • offset (Number) — the offset to start writing into the buffer at
    • length (Number) — an upper bound on the length of the string that we can write

    Returns:

    • (Number) — number of bytes written into the buffer
