    Constructor Details

    - (void) constructor(item_type, size, mode = 0x1b6, atime = (new Date()), mtime = (new Date()), ctime = (new Date()))

    Provides information about a particular entry in the file system.

    Parameters:

    • item_type (Number) — type of the item (FILE, DIRECTORY, SYMLINK, or SOCKET)
    • size (Number) — Size of the item in bytes. For directories/symlinks, this is normally the size of the struct that represents the item.
    • mode (Number) — Unix-style file mode (e.g. 0o666)
    • atime (Date?) — time of last access
    • mtime (Date?) — time of last modification
    • ctime (Date?) — time of creation
