    + (void) chdir(dir)

    Changes the current working directory.

    Note: BrowserFS does not validate that the directory actually exists.

    Examples:

    Usage example 

    console.log('Starting directory: ' + process.cwd());
process.chdir('/tmp');
console.log('New directory: ' + process.cwd());
